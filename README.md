# House Price Predictor using Linear Regression

## Project Overview
This repository contains a foundational Machine Learning pipeline built to understand the mathematical principles of Ordinary Least Squares (OLS) and simple linear regression. The model trains on a custom real estate dataset to map the relationship between a house's square footage and its market valuation.

## Tech Stack
* **Language:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, Matplotlib

## Key Metrics
* **Algorithm:** Linear Regression
* **Evaluation Metric:** Mean Absolute Error (MAE) ~2.15 Lakhs
